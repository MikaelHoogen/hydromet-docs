# Klimatprediktorbaserad IDF

Status: Framtida jämförelsemetod

## Syfte

Den klimatprediktorbaserade IDF-modulen ska på sikt kunna jämföra traditionella IDF-trösklar mot mer plats- och klimatscenariobaserade metodfamiljer.

## Princip

Modulen ska inte ersätta första IDF-kärnan.

Den ska användas som:

```text
jämförelsemetod
framtidslager
metodosäkerhet
scenarioanalys
```

## Möjliga metadata

```text
plats / gridruta
scenario / RCP
tidsperiod
klimatprediktorer
klimatfaktor
metodversion
beräkningsregn
```

## Designregel

```text
Resultat ska presenteras som metodjämförelse, inte som exakt framtida sanning.
```
