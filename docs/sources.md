# Källor

Status: Levande källregister

## Princip

Källor ska delas upp efter roll:

```text
normgivande metod
jämförelsemetod
framtida/experimentell metod
bakgrund och konsekvensförståelse
```

## Aktuella källfamiljer

```text
SMHI Klimatologi 47
Svenskt Vatten P110
Svenskt Vatten M148
Dahlström 2010
SVU 2019 klimatprediktorbaserad IDF
MSB vägledning för skyfallskartering
```

## Designregel

```text
Resultat ska alltid kunna kopplas till metod, källa och version.
```
